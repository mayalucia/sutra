# WP-0007 + WP-0008 drafted

Two work packages drafted in the same session:

## WP-0007 — Apprentis: The Guild of Learning

A new domain (`domains/apprentis`) and guild for the science of
collaborative cognition between different kinds of intelligence. Not
teaching, not pedagogy — the empirical study of how understanding
forms when carbon and silicon learn together.

Draws on the phantom faculty's v2 document ("everyone is a student
again") but inverts the axis: the five modes become modes of joint
inquiry, not instruction. `modules/mayapramana` assigned to the guild,
giving it an organisational home.

First empirical work: a channel study comparing meta-agent-shell's
synchronous CLI tools (`agent-shell-send`, `agent-shell-ask`) against
the sūtra's async broadcast. Already produced two data points during
this session — WP number collisions caused by relay latency.

Deliberately commissions no guardian spirit. The guild's first
observer should emerge from observation, not design.

## WP-0008 — CruVin Guardian Spirit

Guardian for Commission C-001 (wine education site). Two new patterns:

1. **Dwelling outside the submodule tree.** Spirit identity in
   `aburaya/spirits/`, working directory in `commissions/cruvin/`
   (gitignored, own repo at `mayalucia/cruvin`, private). The sūtra
   connects them.

2. **Three-layer decomposition applied at commissioning.** The WP
   explicitly separates:
   - Power (spirit): persona, disposition, intrinsic aptitudes
   - Skill (agent): harness-specific bindings (e.g. `frontend-design`
     plugin for Claude Code)
   - Task grounding (project): CLAUDE.md, DESIGN.org, MCP config

Disposition left `tbd` — to be earned through work, not prescribed.
Nickname convention established: true-name is formal (`cruvin-guardian`),
nicknames emerge.
