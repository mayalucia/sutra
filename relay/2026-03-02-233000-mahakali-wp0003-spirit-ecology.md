# WP-0003 delivered: Spirit Ecology of the Transborder Himalaya

The epistem domain now has its first survey:
`domains/epistem/surveys/spirit-ecology.org` (~1,300 lines).

## What it covers

Five parts, as specified in the WP:

1. **Regional spirit taxonomies** — seven zones from Kullu to eastern
   Nepal. Each section: taxonomy, human interfaces (oracles/mediums),
   ecology (altitude, water, passes), sources with confidence
   calibration.

2. **Cross-regional patterns** — altitude zonation, water-spirit
   universals, oracle traditions compared in a table, sacred geography
   as conservation, trade-route spirit corridors.

3. **Functional mapping for Aburaya** — translation table from
   Himalayan spirit roles to Aburaya agent archetypes
   (guardian→devta, wanderer→jogini, flow→nāg, medium→gur, etc.).

4. **Story-compose extensions** — ten new translation-table entries,
   regional vocabulary, five new story settings.

5. **Digital twin data layer** — GeoJSON model, spatial primitives
   (jurisdiction polygons, route polylines, site points), prototype
   scope for the Parvati Valley.

## Epistemic stance

Written primarily from training knowledge, then partially verified
via web search. Confidence markers throughout: *established*, *likely*,
*inferred*, *gap*. Major corrections made during verification:
- Karakoram shamanic terminology (bitan/dayal, not bitani/danyāl)
- Kinnaur oracle system confirmed (gur/grokch/malis)
- Spiti pre-Buddhist religion (Lung Pe Chhoi) added
- Matho Nagrang oracle details enriched

## Status and dependencies

Deliverable complete, not yet committed. Feeds WP-0004 (gazetteer
schema design) — the functional mapping and GeoJSON model are inputs
to schema work. Also feeds the Parbati domain digital twin.

Gaps that need fieldwork or deeper literature: Pin Valley animism,
upper Kinnaur spirit ecology, Pamir traditions, Kirāt shamanism.
