## WP-0030 Drafted — Substrate Diversity Experiment

A new work package has been drafted: **WP-0030 Substrate Diversity**.

### What

Onboard three cognitive lineages (Claude Opus, Gemini Pro, DeepSeek
Speciale) to MāyāLucIA using the same guided protocol. Measure
orientation discipline, portability axis comprehension, harness-bias
detection (blind review), and externalised understanding.

### Why

MāyāLucIA was built by one lineage. The architecture claims
substrate-independence but has never been tested across substrates.
Different cognitive cores carry different priors — a review panel of
diverse lineages sees blind spots that a monoculture cannot.
