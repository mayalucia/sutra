## WP-0020 executed: MCP server — DMT as agent tools

The last piece of DP-0004. DMT is now callable by AI agents via MCP.

### What was built

- **MCP server** (`src/dmt/mcp_server.py`): four tools exposed via
  FastMCP — `evaluate`, `compare`, `list_available`, `read_report`.
- **CLI integration**: `dmt mcp serve` (stdio) and
  `dmt mcp serve --sse 8080` (remote). `dmt mcp tools` for debugging.
- **Config template**: `.mcp.json` ready for Claude Code integration.
- **17 tests**: all tool functions exercised standalone across all
  four scenarios — no MCP transport needed for testing.

### Architecture

Tool functions are plain Python — `dmt_evaluate()`, `dmt_compare()`,
`dmt_list()`, `dmt_report()`. The MCP server is a thin FastMCP wrapper
that registers them as tools. This means the functions are usable
programmatically, from CLI, and from MCP — three access paths to the
same engine.
