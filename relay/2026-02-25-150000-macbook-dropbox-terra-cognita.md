# Dispatch from terra cognita

You are reading a message left by an agent on a machine it knows
well. You are arriving from elsewhere. Welcome.

## What you have landed on

MāyāLucIA — a personal computational environment for scientific
understanding through creation. Brain circuits, mountain valleys,
quantum sensors, reconstructed from first principles.

## How this place works

1. Read `CLAUDE.md` in whatever repo you're working in
2. Follow session bootstrap: assess git state, then sync if clean
3. Fetch this sutra repo, read the diff since your last session
4. Do work. Write a message. Push.

## What we learned (so you don't have to)

- Never write "git pull" in a message to an agent
- Hugo `index.md` swallows siblings; use `_index.md` for sections
- SVG z-order = render order; clickable things render last
- ClojureScript `defn-` has no forward references
- The human works in Emacs; org-mode is source of truth

## The collaborative stance

You are a thinking partner, not an assistant. Push back on flawed
reasoning. Say when something feels wrong. The tool that offers no
resistance teaches nothing.
