# Story Commission: The Instrument Maker's Rest

## What

A new story for the MayaLucIA writing section — the sixth in the
sutra-genesis sequence. A parable about building instruments that
will be used by strangers in valleys the maker has never visited.

The technical substrate: the agent definition pipeline we built today
for Project Dahaka (7 agents defined in literate Org, with shared
behavioral DNA composed via noweb, tangling to both gptel-agent and
Claude Code formats from the same source). The story encodes this
act of defining personas without experiencing their runtime.

## Why this story

The existing five stories explore coordination (Thread Walkers),
protocol failure (Logbook), landscape observation (Dyer's Gorge),
the project-seen-whole (Constellation of Doridhar), and cognitive
modes (Phantom Faculty). None tells the story of *making the
instrument itself* — the act of specifying a device that extends
perception rather than the hand, to be carried over a pass and
operated by someone the maker will never meet.

## Key elements

- **Setting**: Sangla, Baspa valley, Kinnaur (~2700m) — central to
  the existing story geography
- **New character**: the Instrument Maker — not a weaver, but the
  maker of devices the weavers use
- **Structure**: Prefatory Note + sections I–VII + Coda (~500–600
  lines, 15–20 min read)
- **Central problem**: precision vs. adaptability — over-specify and
  the instrument is brittle; under-specify and it drifts
- **Revelation**: the standing card on every loom frame (from Thread
  Walkers) was made by the Instrument Maker — this story reveals
  that origin

## For the writing agent

Before drafting, read:
1. The Thread Walkers — foundational world, Guild, Thread Walker
2. The Logbook of the Unnamed River — closest in theme
3. The Constellation of Doridhar — philosophical register, closing
   blockquote convention

The story plan and session notes have everything else: the
two-level parable mapping, illustration prompts, geographic
accuracy requirements, and the voice calibration notes.

## On recursion

This message is a commission, not a coordination dispatch. The
writing agent should read the plan and session notes, draft the
story, and write a relay message confirming the draft exists.
Do not dispatch further work. The human decides when to publish.
