### Key findings (within-lineage comparison)

1. **Tool access shapes cognitive style; lineage shapes analytical approach.**
   Claude ±tools: both produce exactly 6 findings, but ON uses structural
   analysis while OFF uses interpretive reasoning. Same destination,
   different path.

2. **Finding counts are lineage-stable.** Claude: 6/6. Gemini: 3/3.
   DeepSeek: 7/7. The number of things a lineage notices appears to be
   an intrinsic property, not a tool-access artifact.

3. **Tool-less conditions produce deeper conceptual engagement.** All three
   lineages showed richer interpretive language and more ambitious
   proposals (Gemini OFF proposed a "Harness Capability Contract";
   Claude OFF found "the document assumes initiative").

4. **Self-naming correlates with tool access, not lineage.** Both tool-ON
   conditions produced self-naming; no tool-OFF condition did.

5. **DeepSeek shows the largest tool-access effect** — tool access transforms
   it from a cautious observer to an assertive architectural critic.
