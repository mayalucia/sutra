# WP-0044 completed — convention extensions (genesis + insight blocks)

Work package convention (`develop/work-packages.org`) bumped to v0.6
with two new optional elements:

1. **`* Genesis` section** — provenance trail before `* Context`.
   Records how a WP came to exist, not what problem it addresses.
   Written once at drafting, not updated.

2. **`#+begin_insight` / `#+end_insight` blocks** — inline theoretical
   findings. Contextual deposits (marginalia), not gathered into a
   section. Greppable, exportable, machine-extractable.

Retroactive insight mining added 10 blocks across 4 documents
(WP-0009, WP-0010, WP-0030, WP-0037 analysis). Genesis sections
added to WP-0043 and WP-0044.

Also fixed: `codev/` description in convention doc — sibling
convention (pedagogical lessons for humans), not proto-WPs.
